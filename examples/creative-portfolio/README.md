# Creative Portfolio Example

A modern, sleek portfolio website for photographers, designers, and creative professionals.

## Features

- **Project Grid** - Filterable portfolio with categories
- **Lightbox Viewer** - Click to expand project details
- **About Section** - Professional bio with stats
- **Testimonials** - Client reviews
- **Dark Theme** - Modern black with vibrant accents
- **Smooth Animations** - Framer Motion interactions

## Color Scheme

- **Dark:** `#0a0a0a` (background)
- **Accent Coral:** `#ff6b6b`
- **Accent Teal:** `#4ecdc4`
- **Accent Yellow:** `#ffe66d`
- **Accent Purple:** `#a8dadc`

## Quick Start

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
vercel
```

## Customization

- Update projects in `src/components/ProjectGrid.jsx`
- Modify bio in `src/components/About.jsx`
- Change testimonials in `src/components/Testimonials.jsx`
- Update contact info in `src/components/Contact.jsx`
- Adjust colors in `tailwind.config.js`

## Features

- **Category Filtering** - Filter projects by type
- **Animated Grid** - Smooth layout transitions
- **Lightbox Modal** - Full-screen project viewing
- **Gradient Backgrounds** - Vibrant, eye-catching colors
