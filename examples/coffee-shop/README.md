# Coffee Shop Example

A warm, inviting one-page website for a local coffee shop.

## Features

- **Menu Section** - Coffee & pastries with pricing
- **Gallery** - Visual showcase of products and atmosphere  
- **Hours & Location** - Business information and contact
- **Cozy Design** - Brown, cream, and coffee-inspired colors
- **Mobile Responsive** - Optimized for all devices

## Color Scheme

- **Coffee Dark:** `#2d1f16`
- **Espresso:** `#4a3728`
- **Mocha:** `#8b7355`
- **Accent Warm:** `#d4a574`
- **Cream:** `#f5f1e8`

## Quick Start

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
vercel
```

## Customization

- Update menu items in `src/components/Menu.jsx`
- Modify hours in `src/components/Hours.jsx`
- Change contact info in `src/components/Contact.jsx`
- Adjust colors in `tailwind.config.js`
