# Wellness Studio Example

An earthy, calming website for holistic practitioners and wellness professionals.

## Features

- **Services Showcase** - Healing modalities with pricing
- **About Section** - Practitioner bio and credentials
- **Testimonials** - Client reviews and social proof
- **Booking CTA** - Clear calls-to-action
- **Natural Design** - Sage green, lavender, earthy tones

## Color Scheme

- **Sage:** `#5a8f5a` (main)
- **Lavender:** `#9a7abe` (accent)
- **Earth Cream:** `#fdfcf9`
- **Earth Sand:** `#f5f1eb`

## Quick Start

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
vercel
```

## Customization

- Update services in `src/components/Services.jsx`
- Modify practitioner info in `src/components/About.jsx`
- Change testimonials in `src/components/Testimonials.jsx`
- Adjust booking details in `src/components/Booking.jsx`
- Customize colors in `tailwind.config.js`
